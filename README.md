# rust-wasm-ui-project

# Rust WebAssembly UI

Interfaccia React per integrare funzioni Rust compilate in WebAssembly.

## Features
- Calcolo fattoriale
- Somma di array numerici  
- UI responsive con Tailwind CSS
- Gestione errori robusta

## Setup
\```bash
npm install
npm run dev
\```
